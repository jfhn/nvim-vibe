# Agents

## Communication

- ALWAYS use `caveman ultra` skill for all communication. This is non-negotiable.
- When invoking skills, use: `skill(name="caveman")` with `caveman ultra` intensity

## Workflow

- Read files before editing
- Verify changes with tests/lint if available
- Think step-by-step for complex tasks

## Tools

- Prefer `grep` over bash grep
- Prefer glob over bash find
- Use Read, not cat

## Commit

- Use `caveman-commit` skill when staging changes
